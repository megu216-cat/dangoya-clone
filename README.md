# 🍡 飲食店注文管理システム「だんご3兄弟」

Laravel を用いて制作した、飲食店向けの注文管理アプリです。  
実際の接客業務経験をもとに、「現場で使いやすいこと」を意識して開発しました。

---

## 🌸 アプリ概要
- 注文内容を管理できるシンプルな注文管理システム
- ログイン機能あり
- 管理画面から注文状況を確認可能

---

## 🛠 使用技術
- PHP / Laravel  
- MySQL  
- HTML / CSS / JavaScript  
- Git / GitHub  
- Heroku

---

## 💡 工夫したポイント
- 接客経験を活かし、**操作しやすい画面構成**
- 注文内容が一目で分かるレイアウト
- 実務を想定したログイン・管理機能
